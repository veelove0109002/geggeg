1. **CSS对齐技巧**：
   - 使用 `inline-flex` 而非 `flex` 避免块级元素问题
   - 同时设置 `vertical-align:middle` 确保垂直居中
   - 统一 `line-height` 确保行高一致
   - 移除默认的 `padding` 和 `margin`

2. **弹窗设计**：
   - 使用 LuCI 的 `ui.showModal()` API
   - 红色警告主题（#dc2626, #fee2e2, #fef2f2）

3. **按钮样式**：
