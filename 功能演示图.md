## 📊 数据流转图

### 前端到后端的数据流

```
┌─────────────────────────────────────────────────────────────┐
│                         浏览器 (前端)                        │
├─────────────────────────────────────────────────────────────┤
│  1. 用户点击上报按钮                                         │
│     └→ reportIcon('luci-app-xxx')                          │
│                                                             │
│  2. 收集数据                                                │
│     • package: 'luci-app-xxx'                              │
│     • comment: '用户输入的问题描述'                        │
│                                                             │
│  3. 构建 POST 请求                                          │
│     URL: /admin/vum/uninstall/report_icon                  │
│     Body: package=xxx&comment=xxx                          │
│                                                             │
│  4. 发送请求 (AJAX)                                         │
│     └→ _httpJson(url, {method:'POST', body:...})          │
└──────────────────┬──────────────────────────────────────────┘
                   │ HTTP POST
                   ▼
┌─────────────────────────────────────────────────────────────┐
│                     OpenWrt 设备 (后端)                      │
├─────────────────────────────────────────────────────────────┤
│  5. LuCI 接收请求                                           │
│     └→ action_report_icon()                                │
│                                                             │
│  6. 解析参数                                                │
│     • package = http.formvalue('package')                  │
│     • comment = http.formvalue('comment')                  │
│                                                             │
│  7. 收集设备信息                                            │
│     • hostname = sys.hostname()                            │
│     • model = uname -m                                     │
│     • system = uname -s                                    │
│                                                             │
│  8. 构建 JSON 数据                                          │
│     {                                                       │
│       "package": "luci-app-xxx",                           │
│       "comment": "...",                                    │
│       "timestamp": 1704067200,                             │
│       "device_info": {...}                                 │
│     }                                                       │
│                                                             │
│  9. 写入临时文件                                            │
│     └→ /tmp/icon_report_data.json                         │
│                                                             │
│ 10. 发送到远程服务器                                        │
│     优先: curl -X POST -d @file url                        │
│     回退: wget --post-file=file url                        │
│     最后: uclient-fetch --post-file=file url               │
└──────────────────┬──────────────────────────────────────────┘
                   │ HTTPS POST
                   ▼
┌─────────────────────────────────────────────────────────────┐
│                   远程服务器 (tb.vumstar.com)            │
├─────────────────────────────────────────────────────────────┤
│ 11. 接收上报数据                                            │
│     POST /api/report/icon                                  │
│                                                             │
│ 12. 保存到数据库                                            │
│     INSERT INTO icon_reports ...                           │
│                                                             │
│ 13. 发送通知 (可选)                                         │
│     • 邮件通知管理员                                        │
│     • 钉钉/企业微信 Webhook                                 │
│                                                             │
│ 14. 返回响应                                                │
│     {                                                       │
│       "ok": true,                                          │
│       "message": "Report received"                         │
│     }                                                       │
└──────────────────┬──────────────────────────────────────────┘
                   │ JSON Response
                   ▼
┌─────────────────────────────────────────────────────────────┐
│                     OpenWrt 设备 (后端)                      │
├─────────────────────────────────────────────────────────────┤
│ 15. 接收响应                                                │
│     解析 JSON 结果                                          │
│                                                             │
│ 16. 返回给前端                                              │
│     json_response({ok:true, message:...})                  │
└──────────────────┬──────────────────────────────────────────┘
                   │ JSON Response
                   ▼
┌─────────────────────────────────────────────────────────────┐
│                         浏览器 (前端)                        │
├─────────────────────────────────────────────────────────────┤
│ 17. 处理响应                                                │
│     if (res.ok) {                                          │
│       显示成功通知                                          │
│     } else {                                               │
│       显示失败通知                                          │
│     }                                                       │
│                                                             │
│ 18. 关闭对话框                                              │
│     ui.hideModal(modal)                                    │
└─────────────────────────────────────────────────────────────┘
```

## 🎬 动画效果

### 按钮悬停动画

```javascript
// 鼠标进入
scale: 1.0 → 1.15 (0.15s ease)
background: #ff6b6b → #fa5252
box-shadow: 0 1px 3px → 0 2px 6px

// 鼠标离开
scale: 1.15 → 1.0 (0.15s ease)
background: #fa5252 → #ff6b6b
box-shadow: 0 2px 6px → 0 1px 3px
```

### 对话框出现动画

```javascript
// 淡入 + 缩放
opacity: 0 → 1 (0.2s ease)
transform: scale(0.95) → scale(1.0)

// 背景遮罩淡入
background: rgba(0,0,0,0) → rgba(0,0,0,0.5)
```
