# luci-app-uninstall 快速使用指南

## 🎯 新功能概览

### 1️⃣ 日志折叠功能
卸载和升级弹窗的日志默认折叠，点击按钮可展开查看详细信息。

### 2️⃣ 批量卸载功能
可以一次性选择多个软件包进行批量卸载，大幅提升操作效率。

---

## 📖 使用教程

### 单个卸载（原有功能）

1. 找到要卸载的软件包
2. 设置卸载选项（删除配置、卸载依赖、清空缓存）
3. 点击"卸载"按钮
4. 确认卸载
5. 查看进度（日志默认折叠，可点击"展开日志"查看详情）

### 批量卸载（新功能）

#### 方式一：手动选择
```
1. 勾选要卸载的软件包（点击卡片左侧的复选框）
2. 工具栏显示"已选: X"
3. 点击"批量卸载"按钮
4. 确认卸载列表
5. 查看批量卸载进度
6. 完成后查看统计结果
```

#### 方式二：使用全选
```
1. 点击工具栏的"全选"复选框
2. 所有显示的软件包被选中
3. 取消不需要卸载的包（点击对应复选框）
4. 点击"批量卸载"按钮
5. 确认并执行
```

#### 方式三：搜索后批量卸载
```
1. 在搜索框输入关键词（如"test"）
2. 筛选出相关软件包
3. 点击"全选"或手动选择
4. 批量卸载选中的包
```

## ⚠️ 注意事项

### 批量卸载
- ❌ **不可撤销**：批量卸载后无法恢复，请谨慎操作
- ⚠️ **依赖关系**：注意软件包之间的依赖关系
- 🔒 **系统包**：不建议批量卸载系统默认插件
- 🌐 **网络连接**：保持网络连接稳定

### 卸载选项
- **删除配置文件**：配置将永久丢失
- **卸载相关依赖**：可能影响其他软件包
- **清空插件缓存**：清除插件的缓存数据

## 🔧 常见问题

### Q1: 批量卸载按钮是灰色的？
**A**: 需要先勾选至少一个软件包，按钮才会启用。

### Q2: 全选后如何取消部分选择？
**A**: 直接点击不需要卸载的软件包的复选框即可取消。

### Q3: 批量卸载中途失败怎么办？
**A**: 单个包失败不影响其他包，最终会显示成功和失败的统计。展开日志可查看详细错误信息。

### Q4: 如何查看卸载详情？
**A**: 点击"展开日志"按钮，可以看到每个软件包的卸载过程和结果。

### Q5: 批量卸载的顺序是什么？
**A**: 按照选择的顺序依次卸载，一个完成后再卸载下一个。
