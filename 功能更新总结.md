# 上报图标功能 - 最新更新

## 📍 更新内容

### 1. 上报按钮位置调整
- **之前**: 按钮位于图标右下角(红色圆形 "!" 图标)
- **现在**: 按钮位于**卡片左下角**(红色标签样式,显示 "! 上报图标")

### 2. 排除自身应用
- **luci-app-uninstall** 自身的卡片**不显示**上报按钮
- 只有其他应用才显示上报功能

### 3. 样式优化

#### 按钮外观
```
┌─────────────────────────────────────┐
│  [图标] 应用名称                      │
│         Package Name                │
│  [ ] 删除配置  [ ] 卸载依赖           │
│                                     │
│  [! 上报图标]              [卸载]    │ ← 左下角
└─────────────────────────────────────┘
```

#### 如果有 VUM-Plugin 标签
```
┌─────────────────────────────────────┐
│  [图标] 应用名称                      │
│         Package Name                │
│  [ ] 删除配置  [ ] 卸载依赖           │
│                                     │
│  [VUM-Plugin] [! 上报图标]  [卸载]   │ ← 自动调整位置
└─────────────────────────────────────┘
```

### 4. 交互效果
- 🎯 鼠标悬停: 按钮上浮并变亮
- 🎨 颜色: `#ff6b6b` → `#fa5252` (悬停)
- ✨ 阴影增强,视觉反馈更明显

## 🔧 技术细节

### 位置计算
```javascript
// 如果有 VUM-Plugin 标签,向右偏移 100px
// 否则从左边 12px 开始
left: (pkg.vum_plugin ? '100px' : '12px')
```

### 排除逻辑
```javascript
// 仅当应用名称不是 luci-app-uninstall 时显示按钮
if (pkg && pkg.name !== 'luci-app-uninstall') {
    // 创建上报按钮
}
```

### 按钮样式
```css
position: absolute;
left: 12px (或 100px);
bottom: 6px;
height: 20px;
padding: 0 8px;
background: #ff6b6b;
border-radius: 10px;
font-size: 11px;
color: #fff;
```

## 📦 部署方式

### 方式一: 完整重新编译
```bash
cd openwrt
make package/luci-app-uninstall/clean
make package/luci-app-uninstall/compile V=s
```

### 方式二: 仅更新 JS 文件(开发测试)
```bash
# 直接传输 JS 文件到设备
scp luci-app-uninstall/htdocs/luci-static/resources/view/uninstall/main.js \
    root@192.168.1.1:/www/luci-static/resources/view/uninstall/

# 清除浏览器缓存后访问
# 通常按 Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
```

### 方式三: 完整安装包
```bash
# 编译
make package/luci-app-uninstall/compile V=s

# 传输到设备
scp bin/packages/*/luci/luci-app-uninstall_*.ipk root@192.168.1.1:/tmp/

# 在设备上安装
ssh root@192.168.1.1
opkg remove luci-app-uninstall
opkg install /tmp/luci-app-uninstall_*.ipk
/etc/init.d/uhttpd restart
```
