# luci-app-uninstall 功能更新总结

## 更新日期
2025-11-06

## 更新内容

### 1. 日志折叠功能 ✅
**功能描述**：卸载和升级弹窗的日志默认折叠，用户可自由选择展开

**主要特性**：
- 日志默认折叠（高度为0）
- 添加"展开日志"/"折叠日志"切换按钮
- 平滑的展开/折叠动画（0.3秒过渡）
- 展开时按钮变蓝色，折叠时为灰色
- 日志实时更新，自动滚动到底部

**影响范围**：
- 卸载弹窗（uninstall 函数）
- 升级弹窗（doUpgrade 函数）

### 2. 批量卸载功能 ✅
**功能描述**：用户可以多选软件包后进行批量卸载

**主要特性**：
- 批量操作工具栏（全选、已选数量、批量卸载按钮）
- 每个软件包卡片添加复选框
- 全选/取消全选功能
- 半选状态支持（部分选中时）
- 批量卸载确认对话框
- 批量卸载进度显示
- 实时进度条和状态更新
- 成功/失败统计

**用户流程**：
1. 勾选要卸载的软件包
2. 点击"批量卸载"按钮
3. 确认卸载列表
4. 查看实时进度
5. 完成后查看统计结果

## 使用建议

### 适用场景
1. **批量清理**：一次性卸载多个不需要的插件
2. **测试环境**：快速清理测试安装的插件
3. **系统优化**：批量移除冗余软件包
4. **版本升级**：批量卸载旧版本插件

### 注意事项
1. 批量卸载不可撤销，请谨慎操作
2. 注意依赖关系，避免误删系统包
3. 建议先展开日志查看详细信息
4. 保持网络连接稳定
