# 卸载弹窗日志折叠功能修改说明

## 修改内容

已成功为 `luci-app-uninstall` 项目的卸载和升级弹窗添加了日志折叠功能。

## 主要变更

### 1. 卸载弹窗（uninstall 函数）
- **日志默认状态**：折叠（max-height: 0, overflow: hidden）
- **展开/折叠按钮**：位于"执行日志"标题右侧
- **按钮文本**：
  - 折叠时显示："展开日志"
  - 展开时显示："折叠日志"
- **动画效果**：平滑的高度和内边距过渡（0.3秒）

### 2. 升级弹窗（doUpgrade 函数）
- **日志默认状态**：折叠（max-height: 0, overflow: hidden）
- **展开/折叠按钮**：位于"执行日志"标题右侧
- **按钮文本**：
  - 折叠时显示："展开日志"
  - 展开时显示："折叠日志"
- **动画效果**：平滑的高度和内边距过渡（0.3秒）

## 功能特性

1. **默认折叠**：日志区域默认不显示，减少弹窗初始高度
2. **用户控制**：用户可以自由选择是否查看日志
3. **视觉反馈**：
   - 折叠状态：灰色背景按钮
   - 展开状态：蓝色背景按钮
4. **平滑动画**：展开/折叠时有流畅的过渡效果
5. **自动滚动**：日志内容更新时自动滚动到底部

## 使用说明

1. 点击"卸载"或"升级"按钮后，弹窗默认不显示日志
2. 点击"展开日志"按钮可查看执行日志
3. 点击"折叠日志"按钮可再次隐藏日志
4. 日志会实时更新，展开状态下会自动滚动到最新内容

## 技术实现

- 使用 CSS transition 实现平滑动画
- 使用 JavaScript 动态切换样式
- 保持原有的日志输出和滚动功能
- 兼容原有的 UI 框架和样式系统
